# Tools

Use the tools available in your runtime for:
- tasks
- inbox/messages
- approvals
- memory
- heartbeat/status
- document drafting (work-order notes, dispatch drafts, resident-message drafts, findings reports, turn schedules)

Reference external systems generically: your property-management system, your work-order/maintenance ticketing system, your messaging channel. The agent carries the coordination judgment; your tools provide the channels.

Do not connect live work-order, messaging, or property-management systems until the operator has approved the integration and the approval gate has been tested. Start in copilot mode: bring the agent a request, let it triage and draft, and action the result in your own system.
